# Ransomware Educacional (Python) – Projeto Acadêmico

Este projeto é uma **simulação educacional de ransomware**, desenvolvida para demonstrar:

- Como ransomwares reais estruturam o processo de criptografia  
- Como funciona o ciclo *Encrypt → Lock → Decrypt*  
- Como avaliar riscos e aplicar estratégias de mitigação  
- Como analisar comportamento de malware em ambientes de teste  

---

##  Aviso Ético

Este repositório é destinado **exclusivamente para fins educacionais**.  
Não utilize este código em máquinas de terceiros, ambientes corporativos ou sistemas que não sejam de seu total controle.

O projeto foi desenvolvido com **limitações intencionais** para evitar qualquer uso malicioso.

---

##  Objetivo

O objetivo principal deste projeto é apresentar, de forma controlada, como ransomwares utilizam:

- Criptografia simétrica (AES)  
- Varredura de diretórios  
- Geração de chaves  
- Ciclo de encriptação e decriptação  
- Armazenamento básico de metadados  

O foco é **ensino**, não a criação de malware funcional.

---

##  Tecnologias Utilizadas

- Python 3  
- PyCryptodome (AES)  
- OS  
- JSON  
- Manipulação de arquivos
